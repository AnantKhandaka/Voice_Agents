# AI Agents Internship Projects

This repository contains AI agent projects developed by our internship team.

## Repository Structure

Each intern has their own folder containing their projects.

## Contributing Guidelines

To add your project:

1. Create a folder with your name
2. Add your source code files
3. Include requirements.txt with dependencies
4. Create a README.md in your folder
5. Add .env.example (never commit actual .env files)
6. Commit and push your changes

## Important

- Never commit API keys or passwords
- Use .env.example to show required environment variables
- Test your code before committing
