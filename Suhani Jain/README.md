# Voice RAG Assistant

Voice-controlled RAG system for EHR documents using LangChain and Deepgram.

## Projects

1. Main Voice Agent - main_voice_agent.py
2. EHR Tool - ehr_tool.py
3. RAG Engine - rag_engine.py
4. Speech to Text - speech_to_text.py
5. Text to Speech - text_to_speech.py

## Setup

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Create .env file with:
   ```
   DEEPGRAM_API_KEY=your_key
   ```

## Usage

Run the voice assistant:
```
python main_voice_agent.py
```
